# Система технической поддержки для сбора и обработки данных

##  Цель
Разработать систему приёма данных от трёх типов пользователей через три канала, с трёхуровневой технической поддержкой и автоматическими уведомлениями.

---

## Как запустить

1. Клонируйте репозиторий
2. Убедитесь, что у вас установлены Docker Dekstop
3. Выполните: docker-compose up --build
4. Система будет доступна по адресу: http://localhost:8000
5. Для демо-данных: docker-compose run --rm web python manage.py create_demo_data

## Архитектура

### Пользователи
- **Респонденты** — отправляют данные
- **Администраторы** — управляют сроками и мониторингом
- **Поставщики** — интегрируют API

### Каналы приёма данных
1. **API** 
2. **Онлайн-форма** 
3. **Оффлайн-файл**
